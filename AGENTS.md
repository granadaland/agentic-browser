# Project Agent Instructions

## BrowserOS 2 Project Ops

For BrowserOS 2 roadmap work, phase planning, feature intake, prioritization, or major implementation requests:

1. Read `.claude/skills/browseros2-project-ops/SKILL.md`.
2. Treat `docs/browseros2/PROJECT-OPS.md` as the source of truth for plan, tasks, progress, phases, and blockers.
3. Map new feature requests to the existing roadmap before implementing them.
4. Update `docs/browseros2/PROJECT-OPS.md` after meaningful progress, reprioritization, or newly discovered blockers.

## Documentation Images

When updating documentation that involves new screenshots or images:

1. Prompt the user to copy the image to their clipboard.
2. Run `python scripts/save_clipboard.py <target_path>`.
3. Save images into the relevant `docs/` path.
